# Мои заметки

## Git 

git config --global user.name "..." - настраиваем имя в гите на компе 

git init - создает отслеживание в папке 

git add . - добавляет все не отслеживаемые файлы в отслеживание 

git status - видим что отслеживается

## Go 

Go (Golang) выделяется минималистичным синтаксисом, высокой скоростью компиляции, встроенной поддержкой конкурентности (горутины) и отсутствием классического ООП, предлагая композицию вместо наследования
